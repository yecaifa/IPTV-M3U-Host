# IPTV 自动更新（自用）

个人使用的 IPTV M3U 自动更新脚本。

从 `iptv.cqshushu.com` 搜索 IPTV，  
仅使用 **Multicast（组播）** 中状态正常的 IP，  
自动下载并更新 `iptv_latest.m3u` 到 GitHub。

## 使用

运行IPTV脚本.bat
或
python iptv_m3u_get.py
